## vscode快捷键
---
### 命令操作
| 按 Press             | 功能 Function |
| -------------------- | ------------- |
| Ctrl + Shift + P, F1 | 显示命令面板  |
| Ctrl + P             | 快速打开      |
| Ctrl + Shift + N     | 新窗口        |
| Ctrl + Shift + W     | 关闭窗口      |
---
### 基础编辑
| 按 Press                    | 功能 Function   |
| --------------------------- | --------------- |
| Ctrl+X                      | 剪切行(空选定)  |
| Ctrl+C                      | 复制行(空选定)  |
| Alt+↑/↓                     | 向上/向下移动行 |
| Shift+Alt+↑/↓               | 向上/向下复制行 |
| Ctrl+Shift+K                | 删除行          |
| Ctrl+Enter                  | 在下面插入行    |
| Ctrl+Shift+Enter            | 在上面插入行    |
| Ctrl+Shift+\\|跳到匹配的括号 |
| Ctrl+]/[                    | 缩进行          |
| Home                        | 转到行首        |
| End                         | 转到行尾        |
| Ctrl+Home                   | 转到文件开头    |
| Ctrl+End                    | 转到文件末尾    |
